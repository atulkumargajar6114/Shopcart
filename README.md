# ShopCart Online Store

ShopCart is a user-friendly online store website built using Java, MySQL, Hibernate, JSP, HTML, CSS, JavaScript (Basic), and Bootstrap4. The website allows customers to browse, order, and track products, while the business owner can manage the website through an admin panel.

## Features

1. User-Friendly Interface: ShopCart provides a user-friendly interface that makes it easy for customers to browse and order products.

2. Admin Panel: The website includes an admin module that allows the business owner to manage products, categories, and user details, ensuring an up-to-date and organized inventory.

3. Account Creation: Customers can create accounts on the website, enabling them to save their information for future purchases and track their orders.

4. Product Browsing: Customers can easily browse through the products available on the website and view their details.

5. Shopping Cart: The website allows customers to add products to their cart and remove items if needed.

6. Checkout: Customers can review their orders, provide necessary information, and complete the purchase through the secure checkout option.

## Technologies Used

## Getting Started

To set up the ShopCart project locally, follow these steps:

1. Clone the repository: `git clone https://github.com/atulkumargajar6114/Shopcart`

2. Set up the database: Create a MySQL database and import the provided SQL file containing the necessary tables and data.

3. Configure the project: Update the `hibernate.cfg.xml` file with your database credentials.

4. Deploy the project: Deploy the project on a suitable Java web server (e.g., Apache Tomcat).

5. Access the website: Open your web browser and visit `http://localhost:8080/ShopCart` to access the ShopCart online store.
